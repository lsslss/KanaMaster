日文歌詞學習工具：淚光閃閃 (Nada Soso)

這是一個專為日文初學者設計的互動式歌詞學習網頁。透過夏川里美的經典名曲《淚光閃閃》（涙そうそう），使用者可以一邊聽音樂，一邊練習日文五十音、單字與語感。

🌟 功能亮點

逐句教學卡片：將歌詞拆解成單句頁面，專注學習。

漢字假名標注 (Furigana)：所有漢字皆有標注平假名，方便認讀五十音。

AI 語速自訂發音：內建 TTS 發音功能，提供 0.3x 到 1.5x 的語速調整，適合不同階段的學習者。

YouTube 原曲同步：整合 YouTube API，點擊即可聽取該句的原曲演唱片段。

自動跟隨模式 (Sync Mode)：網頁會隨音樂播放進度自動翻頁，實現 KTV 式的學習體驗。

單字解析：每句歌詞下方皆附帶重點單字的詳細解釋。

🚀 快速部署到 GitHub Pages

本專案採用單一 HTML 檔案設計，部署極其簡單：

在 GitHub 建立一個新的儲存庫 (Repository)。

將專案中的 index.html 上傳至該儲存庫。

進入儲存庫的 Settings > Pages。

在 Build and deployment 下方的 Branch 選擇 main，然後點擊 Save。

稍等片刻，即可透過 GitHub 提供的網址訪問你的學習網頁。

我的試用網址 http://lsslss.github.io/KanaMaster/NadaSoso.html

🛠 使用技術

前端框架：Tailwind CSS (樣式設計)

圖示庫：Lucide Icons

音訊處理：Web Speech API (AI 發音)

影音同步：YouTube IFrame Player API

字體：Google Fonts (Noto Sans JP / TC)

📖 如何使用

認讀模式：使用「AI 發音」並將語速調慢（建議 0.5x），跟著朗讀練習五十音。

演唱模式：點擊「原曲演唱」按鈕，感受歌手的發音、連讀與情感起伏。

自動模式：勾選「自動跟隨音樂播放」，讓音樂帶領你瀏覽整首歌曲的歌詞。
