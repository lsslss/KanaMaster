🇯🇵 Kana Master - 五十音漢字演變強化練習器

一款專為行動裝置優化的日語五十音學習工具。結合了漢字演變（萬葉假名）、圖像聯覺（Emoji）與弱點強化系統，幫助初學者更快速、更直覺地記住假名。

本專案為完全使用 Gemini 建立，學習日文的同時，也學習使用 Gemini 。

!

🌟 核心特色

🔤 漢字溯源：標註每個平假名的起源漢字（如：あ 源自「安」），利用漢字背景加速記憶。

🍎 圖像聯覺：為每個音節配對專屬 Emoji 與常用單字，建立視覺與聽覺的連結。

🔊 真人發音：整合瀏覽器 TTS 技術，點擊即可聽取「假名 + 單字」的序列發音。

⚡ 弱點強化模式：可手動標記「不熟」的假名，並開啟專屬模式針對性反覆練習。

📱 行動端優化：採用響應式設計，支援手機全螢幕操作，適合隨身利用碎片時間練習。

💾 自動存檔：弱點標記與設定會自動儲存在本地瀏覽器（LocalStorage），進度不丟失。

🚀 快速開始

本專案採用 No Node.js 架構開發，無需安裝任何開發環境即可運行。

直接使用

下載本專案的 index.html。

使用任何現代瀏覽器（Chrome, Safari, Edge）開啟檔案即可開始練習。

佈署到 GitHub Pages

在 GitHub 建立一個新的 Repository。

將 index.html 上傳至根目錄。

前往 Settings > Pages，選擇 main 分支並儲存。

稍等片刻即可透過 https://<你的帳號>.github.io/<專案名>/ 訪問。

你可以在 https://lsslss.github.io/KanaMaster/ 看到目前的結果。

🛠️ 技術架構

框架: React 18 (透過 CDN 引入，免打包)

樣式: Tailwind CSS

圖示: Lucide Icons

語音: Web Speech API (SpeechSynthesis)

轉譯: Babel Standalone

📖 使用說明

一般練習：進入頁面後即可開始翻轉卡片。

標記弱點：在字卡背面點擊「閃電 ⚡」圖示，該字將被加入弱點清單。

強化模式：當清單中有字時，點擊右上角或設定頁的黃色按鈕，即可開啟「弱點強化模式」。

自訂範圍：在設定頁中可勾選想要練習的特定行數（あ行、か行等）。
