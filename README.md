🇯🇵 Kana Master - 旅遊五十音全面強化版

一款專為行動裝置優化的日語五十音學習工具。結合了漢字演變、旅遊實用單字與雙模式練習系統，幫助旅遊愛好者在學習假名的同時，掌握旅日必備詞彙。

!

🌟 核心特色

🔤 漢字溯源：標註每個平假名與片假名的起源漢字（部首），利用漢字背景加速記憶。

🍣 旅遊情境單字：所有關聯單字皆圍繞「食、衣、住、行、購」，讓學習更具實用性。

🔄 平/片假名全覆蓋：可自由選擇練習平假名、片假名或兩者混和練習。

⚡ 模式快切與自動洗牌：直接在練習頁面切換 循序 (Order) 或 隨機 (Shuffle) 模式，隨機模式下完成每輪後會自動洗牌。

🔄 雙面強化顯示：翻轉卡片後同步顯示羅馬拼音、假名、單字日文、中文意義與字源。

🔊 真人發音：點擊喇叭即可聽取「假名 + 旅遊單字」的完整發音。

💾 設定持久化：所有的練習偏好、範圍選擇與弱點清單皆會儲存在瀏覽器中。

🚀 快速開始

你可以在 https://lsslss.github.io/KanaMaster/ 試用目前的成果。

或下載本專案的 index.html 透過任何現代瀏覽器直接開啟。

📖 學習指引

基礎建立：在設定頁選擇「平假名」+「循序模式」，先建立基礎語感。

片假名攻略：旅遊時外來語極多，專門勾選「片假名」進行特訓對於看菜單極有幫助。

隨機驗收：當你覺得熟悉某個行數時，點擊右下角切換至 Shuffle 模式進行自我測驗。

重點突破：將難記的假名標記為「弱點 ⚡」，利用弱點強化模式攻克它們。
