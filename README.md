🇯🇵 Kana Master - 旅遊五十音 PWA & SRS 強化版

一款專為行動裝置與旅遊愛好者優化的日語學習工具。結合了漢字演變、Anki 間隔重複學習法 (SRS) 以及旅遊實用情境單字，讓你更有系統地掌握假名。

!

🌟 核心特色

📱 PWA 支援 (Progressive Web App)：支援離線使用與手機安裝。將網頁「加入主畫面」後，即可享受如原生 App 般的極速體驗。

🧠 Anki SRS 學習演算法：導入專業間隔重複系統，根據你的記憶強度（重來、困難、良好、簡單）智慧排程，在快遺忘前精準複習。

🍣 旅遊情境單字：單字全數更換為「食、衣、住、行」旅遊實用詞彙，同步學習假名與旅日必備單字。

⚡ 雙模式切換：

循序練習 (Order)：按照五十音圖順序，適合初學新單元。

大腦複習 (Anki)：依遺忘曲線優先複習已到期或不熟的字卡。

🔄 平/片假名全覆蓋：包含基本音、濁音、半濁音與常用拗音。

🔊 自動發音/跟讀：標題列一鍵切換「自動發音」模式，每切換到新卡片即自動帶讀「假名 + 單字」。

💾 資料持久化：所有的學習進度、SRS 因子、今日待複習數量與範圍設定皆儲存在本地瀏覽器中。

🚀 快速開始

線上試用：https://lsslss.github.io/KanaMaster/

安裝 App：在手機瀏覽器開啟後，點擊「加入主畫面」或「安裝應用程式」。

🛠️ 技術架構

前端: React 18 (CDN 免打包架構)

樣式: Tailwind CSS

圖示: Lucide Icons

語音: Web Speech API

PWA: Service Worker (Inline Blob) & Manifest Data URI

📖 學習指引

學新字：在設定勾選範圍後，使用「循序練習 (List 圖示)」建立基礎記憶。

長效記憶：切換至「大腦 (Brain 圖示)」模式，每日進行 Anki 複習，根據反應速度誠實評分。

跟讀特訓：開啟紫色喇叭按鈕，邊看邊讀，強化聽力與口說連結。
