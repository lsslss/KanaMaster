🇯🇵 Kana Master - 五十音漢字演變強化練習器 (模式快切版)

一款專為行動裝置優化的日語五十音學習工具。結合了漢字演變（萬葉假名）、圖像聯覺（Emoji）與雙模式快切系統，幫助初學者更快速、更直覺地記住假名。

!

🌟 核心特色

🔤 漢字溯源：標註每個平假名的起源漢字（如：あ 源自「安」），利用漢字背景加速記憶。

🍎 圖像聯覺：為每個音節配對專屬 Emoji 與常用單字，建立視覺與聽覺的連結。

⚡ 模式快切按鈕：直接在練習頁面切換 循序 (Sequential) 或 隨機 (Random) 模式，無需進入設定。

🔄 自動洗牌邏輯：在隨機模式下，完成每一輪練習後會自動重新隨機排列，實現無縫循環學習。

🔄 雙面強化顯示：翻轉卡片後不僅顯示羅馬拼音，同步顯示單字的日文與字源，全方位強化記憶連結。

🎹 全假名覆蓋：完整收錄基本音、濁音、半濁音及拗音。

🔊 真人發音：整合瀏覽器 TTS 技術，點擊即可聽取「假名 + 單字」的序列發音。

📱 行動端優化：採用響應式設計，支援手機全螢幕操作。

💾 自動存檔：模式偏好、弱點標記與範圍設定會自動儲存在本地瀏覽器（LocalStorage）。

🚀 快速開始

你可以在 https://lsslss.github.io/KanaMaster/ 試用目前的成果。

或下載本專案的 index.html 透過任何現代瀏覽器直接開啟。

📖 操作指引

切換模式：點擊右下角的黑/藍色按鈕，即可在「照順序練」與「隨機考」之間切換。

自動循環：隨機模式下，練完當前選擇的所有字會自動重新洗牌。

標記弱點：在字卡背面點擊「閃電 ⚡」圖示。

自訂範圍：在設定頁中可自由勾選想要練習的行數。
